# TimeAlert

A simple timer on terminal, just run in Mac OSX **only**
一个简易的命令行计时器，目前只支持mac 版本

## Installation

  `gem install time_alert`

## Usage

 倒计时分钟：
 `time_alert min NUMBER` 
 
 关闭提示（默认响很久...）
 `time_alert stop`
